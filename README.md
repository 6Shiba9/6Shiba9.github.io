# 🎨 Figma Portfolio UI (Frontend Project)

A responsive portfolio website built from a Figma design using Tailwind CSS and JavaScript.

## 🚀 Features
- Responsive design (Desktop / Mobile)
- Navigation bar with mobile menu toggle
- Image slider with smooth transition
- Clean and modern UI layout
- Portfolio-style website structure

## 🛠 Tech Stack
- HTML5
- Tailwind CSS
- JavaScript (Vanilla)

## 🎯 Purpose
This project was created to practice:
- Converting Figma design into real web UI
- Responsive layout design
- Basic frontend interaction with JavaScript
